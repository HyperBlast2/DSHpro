# DSHpro

A comprehensive game memory manipulation toolkit for advanced analysis and testing.

## Features

- Game memory offset documentation
- Static class and field offset mappings
- Camera and graphics manipulation support
- Entity and player data structures
- Network entity management

## Files

- **Dash.Pro.exe** - Main application executable
- **DVL.exe** - Driver/utility executable
- **driver.sys** - System driver module
- **all_info.txt** - Comprehensive offset database and structure definitions
- **dash.pro** - Qt project configuration

## Building

### Requirements
- Qt 5.15+ or Qt 6.0+
- C++17 compatible compiler
- CMake or qmake

### Compile
```bash
qmake dash.pro
make
```

## Documentation

Refer to `all_info.txt` for complete offset mappings and structure definitions.
